简介

Longstudio Codegen是基于Mybatis的开源Restful API 代码生成器.

# 特性

  • 国际化支持: 支持i18n
  • 操作简单: 简洁的操作界面
  • 动态数据源: 支持动态数据源切换
  • 方便使用: 易于使用的模板管理
  • 快捷搜索: 方便的树视图搜索操作
  • 代码生成: 快速生成Restful API模板代码
  • 支持多数据源: 包括MySQL、Mariadb、Oracle、SQLite和Postgresql...
