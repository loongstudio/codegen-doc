Codegen

基于Mybatis的开源Restful API 代码生成器.

开启未来 →

简单优雅

Restful API模板代码支持快速一键生成，可保留和查看历史记录，方便快捷

易于使用

拥有易于使用的模板管理界面和数据源管理界面，方便的树视图搜索功能令工作效率翻倍

多数据源

支持多种数据源，包括：MySQL、Mariadb、SQLite和Postgresql
